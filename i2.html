Presenters Wanted

To make it happen we need some people to speak and to run workshops.

This will be a single track event, so we can be flexible about the length of workshops.

If you've got an idea for a 5 minute talk, game or workshop that you think will be inspiring - that's fine. Or if you need it to run for an hour, that's fine too. It can even be only loosely related to UX, as long as it's ultimately going to inspire us all to go on and do better UX work at the end of it.

As the first UX South West event, we're keeping the venue fairly modest. It will be in The Rougemont room in the recently renovated Exeter Library. When set up for workshops it will accommodate about 40 people. So ideas for workshops should allow for groups of up-to this size.

If you've not given a talk or run a workshop before, we'd like to think this will be a good opportunity to start. Send in your ideas and we are happy to discuss further to develop the idea and we'll hopefully be able to offer support on the day for running workshops too if needed.

If you have a proposal or an idea to discuss further, please email
john@uxsw.co.uk

Final submissions for talks & workshops need to be in by mid October.

Brass Tacs

This is the first UX South West event and it's being run as a bootstrapped affair. Ticket prices will, hopefully, cover the venue hire and probably not much else. Much as we know you really deserve paying for offering to give a talk or run a workshop, unfortunately we won't be able to pay presenters this time. You will of course get a free ticket and our sincere thanks, we can't do this without you.

If your workshop idea is going to incur costs (pens, paper, stickies, Haribos ... and suchlike) - let us know and we will try and cover these costs for you.
